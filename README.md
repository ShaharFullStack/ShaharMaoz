# Portfolio Website

A modern, responsive portfolio website built with React, Tailwind CSS, and Framer Motion.

## Features

- Responsive design with glass morphism and neon UI elements
- Animated with Framer Motion
- Interactive background
- Project showcase
- Pages for different sections (Home, Projects, Code, Music)

## How to Run

Follow these steps to run the portfolio website on your local machine:

### Prerequisites

- [Node.js](https://nodejs.org/) (v14 or newer)
- npm (comes with Node.js)

### Installation

1. Open your terminal/command prompt.

2. Navigate to the project directory:
   ```
   cd c:\Codes\port\portfolio
   ```

3. Install the dependencies:
   ```
   npm install
   ```
   This might take a few minutes to complete.

4. Start the development server:
   ```
   npm start
   ```

5. Open your browser and navigate to [http://localhost:3000](http://localhost:3000) to see the website.

### Troubleshooting

If you encounter any issues:

- Make sure Node.js is properly installed (check with `node --version`)
- Try deleting the `node_modules` folder and running `npm install` again
- Check for any error messages in the terminal

## Project Structure

- `App.jsx`: Main application component with routing
- `layout.js`: Layout component with navigation and footer
- `Pages/`: Page components
- `components/`: Reusable UI components
- `Entities/`: Data models
- `utils.js`: Utility functions

## Technologies Used

- React
- React Router
- Tailwind CSS
- Framer Motion
- Lucide React (for icons)
